# Creative Direction Council Agent Skill

Expert creative direction council with 6 advisors for visual identity, branding, and design systems.

## Advisors

- **Massimo Vignelli** - Systems, grids, discipline
- **Aaron Draplin** - Simple, bold, quick logos
- **Michael Bierut** - Strategic thinking, process
- **Paula Scher** - Identity at scale, typography
- **Stefan Sagmeister** - Provocative, personal design
- **David Carson** - Breaking rules, intuition

## Installation

### Claude Code
```bash
/plugin add https://github.com/TOPAiiN/creative-direction-council
```

### Claude.ai
Upload the `creative-direction-council` folder or add via skills panel.

## Usage

Activate when you need help with:

- Logo and identity design
- UI design decisions
- Design systems and tokens
- Brand visual direction
- Presentations and pitch decks
- Marketing materials

### Example Prompts

```
Use creative-direction-council to design my logo
Use creative-direction-council to create a design system
Use creative-direction-council for brand identity guidance
Use creative-direction-council to improve my presentation slides
```
